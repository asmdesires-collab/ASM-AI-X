# ASM-AI-X

**ASM-AI-X** is a governance-first reference architecture for Artificial General Intelligence (AGI).

The project formalizes cognitive models, meta-reasoning, agent swarms, memory hierarchies, and safety invariants,
enabling the design of testable, auditable, and institutionally controllable AGI systems.

This repository contains architectural specifications and governance frameworks.
It does **not** include implementation or deployment code.
